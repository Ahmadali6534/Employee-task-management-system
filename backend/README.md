# Employee Task Management System

A full-stack Employee Task Management System built with FastAPI, PostgreSQL, SQLAlchemy, Alembic, and JWT Authentication.

---

# Features

## Authentication
- User registration
- Login with email/password
- Password hashing using bcrypt
- JWT authentication
- Protected routes
- Role-based authorization

## User Management
- Create users
- View users
- Update users
- Soft delete users
- Email uniqueness validation

## Employee Management
- Create employee profiles
- View employees
- Search employees
- Pagination support

## Task Management
- Create tasks
- Assign tasks to employees
- Update task details
- Update task status
- Delete tasks
- Search tasks
- Filter by status and priority
- Pagination
- Sorting

## File Management
- Upload task attachments
- Download files
- Delete files
- File validation
- Maximum file size 10MB

Supported files:

- PDF
- DOCX
- XLSX
- PNG
- JPG
- TXT

## Dashboard
- Total employees
- Total tasks
- Pending tasks
- Completed tasks
- In-progress tasks
